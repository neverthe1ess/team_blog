# React 기초: 4. 리스트 렌더링 (목록 보여주기)

## 1. 리스트 렌더링이란?

**"데이터 배열을 UI 배열로 변신시키기"**입니다. 📋
게시판에는 게시글이 1개가 아니라 10개, 100개가 있겠죠?
이걸 일일이 하드코딩(`<Post />`, `<Post />`...) 할 수는 없습니다.
자바스크립트의 `map()` 함수를 쓰면 데이터 갯수만큼 자동으로 컴포넌트를 만들어줍니다.

## 2. 사용법 (`.map()`)

```tsx
const posts = ["글 1", "글 2", "글 3"];

return (
  <ul>
    {posts.map((post) => (
      <li>{post}</li>
    ))}
  </ul>
);
```

## 3. 핵심 규칙: Key (주민등록번호) 🔑

리스트를 만들 때 React는 각 아이템을 구분할 수 있는 **고유한 이름표(`key`)**를 요구합니다.
그래야 나중에 "어? 2번째 글만 지워졌네?" 하고 똑똑하게 그 부분만 고칠 수 있거든요.

```tsx
// ❌ 나쁜 예 (인덱스 사용 - 순서 바뀌면 꼬임)
posts.map((post, index) => <li key={index}>{post}</li>);

// ✅ 좋은 예 (고유 ID 사용)
posts.map((post) => <li key={post.id}>{post.title}</li>);
```
