# 🎓 13교시: 페이지네이션 (Pagination) 📄

"글이 100만 개면 어떡하죠?"
한꺼번에 다 불러오면 렉이 걸립니다. 그래서 **책 페이지처럼 나눠서(Pagination)** 보여줘야 합니다.

## 1. 백엔드와의 약속 (Query Parameter) 🤝

- `GET /posts?page=1` : 1페이지 (최신 글 10개)
- `GET /posts?page=2` : 2페이지 (그 다음 10개)

## 2. 응답 구조의 변화 📦

원래는 **[배열]**만 줬지만, 이제는 **{데이터, 정보}**를 줍니다.

```javascript
{
  "data": [ ...글 10개... ],
  "meta": {
    "total": 100,      // 전체 글 개수
    "page": 1,         // 현재 페이지
    "lastPage": 10     // 마지막 페이지 번호
  }
}
```

## 3. 프론트엔드 전략 🧠

1.  **URL 바꾸기**: 숫자를 누르면 `router.push('/?page=2')`로 이동.
2.  **데이터 다시 받기**: `page` 파라미터가 바뀌면 `useEffect`가 다시 실행되게 함.
3.  **버튼 그리기**: `meta.lastPage`를 보고 1부터 10까지 버튼을 만듦.
